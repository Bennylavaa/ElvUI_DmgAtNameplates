# ElvUI DmgAtNameplates

A comprehensive damage and combat text addon for ElvUI that displays floating combat text above nameplates and character frames.

## Features

- **Damage Numbers**: Shows damage dealt and received with customizable animations
- **Healing Text**: Displays healing amounts including overheal detection
- **Combat Events**: Miss, dodge, parry, block, resist, and other combat events
- **Pet Combat**: Separate tracking for pet and minion abilities
- **Spell Icons**: Optional spell icons next to damage numbers
- **Multiple Animations**: Various text animation styles (rising, falling, arc, scatter)
- **Smart Scaling**: Automatically scales small periodic damage
- **Flexible Filtering**: Show/hide different types of events
- **Number Formatting**: Multiple number display formats (1.2k, 1,200, 1200)

## Installation

1. Download and extract to your `Interface/AddOns/` folder
2. Requires ElvUI to be installed and enabled
3. Configure via ElvUI options menu under "DmgAtNameplates"

## Configuration

Access settings through: **ElvUI Config → Plugins → DmgAtNameplates**

- **General**: Basic addon settings, fonts, and animations
- **Damage Display**: Configure outgoing, incoming, and pet damage
- **Healing Display**: Set up healing text and overheal options
- **Miss Events**: Customize dodge, parry, miss text appearance
- **Dispel Effects**: Configure dispel and purge notifications
- **Interrupts**: Set up spell interrupt notifications
